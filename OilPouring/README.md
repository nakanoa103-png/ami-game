# 油分けパズル (Oil Pouring Puzzle)

江戸時代の算術書『塵劫記（じんこうき）』に登場する「油分け算」をシミュレートするロジカルパズルです。

## 🎮 このゲームについて
3つのマスの間で油を移し替え、指定された「目標量」を作り出すことが目的です。
AIの歴史を学ぶ過程で、論理的な思考を整理するために作成しました。

## 🔗 いますぐ遊ぶ
[こちらをクリックしてプレイ](https://nakanoa103-png.github.io/ami-game/OilPouring/)

## ✨ 特徴
- **カスタム問題設定**: デフォルトの（8, 5, 3）以外にも、好きな数値でパズルを生成できます。
- **インストール不要**: ブラウザだけで動作するため、スマホやPCで安全に遊べます。

## 🔒 安全性について
- HTML/JavaScript/CSSのみで構成された静的サイトです。
- 個人情報の収集や外部への通信、ファイルのダウンロードは一切行いません。
