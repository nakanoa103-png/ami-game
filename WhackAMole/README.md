# もぐら叩き (Whack-A-Mole)

シンプルだけど、つい熱くなる。ストレス発散（？）のためのもぐら叩きゲームです。

## 🎮 このゲームについて
NOTEのAI歴史シリーズ第4.5話「箸休め回」のために制作しました。
難しいことは抜きにして、ひたすら画面を叩いてハイスコアを目指してください！

## 🔗 いますぐ遊ぶ
[こちらをクリックしてプレイ](https://nakanoa103-png.github.io/ami-game/WhackAMole/)

## 🔥 実装機能
- **コンボシステム**: 連続で叩くとスコアが跳ね上がる爽快なロジックを搭載。
- **ハイスコア保存**: ブラウザ（localStorage）に記録を保存。自分自身との戦いが楽しめます。
- **視覚フィードバック**: 叩いた瞬間のモグラの表情の変化にこだわりました。

## 🔒 安全性について
- 完全にブラウザ完結型のゲームです。
- ハイスコアの記録はあなたのブラウザ内（localStorage）にのみ保存され、外部に送信されることはありません。
